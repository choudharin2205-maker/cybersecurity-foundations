# Password Hash Cracker

A Python tool that cracks MD5 and SHA256 password hashes using a wordlist attack.

Built as part of my cybersecurity learning journey — Day 2 project.

---

## What it does

- Hashes passwords using MD5 or SHA256
- Cracks hashes by running a wordlist attack using rockyou.txt
- Supports switching between hashing algorithms

---

## How to run it

python cracker.py

---

## What I Learned

- How password hashing works (MD5, SHA256)
- Why MD5 is considered weak
- How dictionary attacks work
- Why common passwords are dangerous even when hashed
